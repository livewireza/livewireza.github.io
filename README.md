# News Dashboard

A static news dashboard powered by GitHub Actions.

## Features

- Pulls RSS feeds every 30 minutes
- Stores data as JSON
- Hosted on GitHub Pages
- No backend required

## Install

```bash
npm install
```

Run manually:

```bash
npm run update
```

GitHub Actions will update automatically.

Enable GitHub Pages by serving the repository (or the `public/` directory if using a build step).
